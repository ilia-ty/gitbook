# 서비스 소개

## 서비스 개요

로이어키트는 국내 변호사 및 법률 전문가를 위해 설계된 올인원 AI 법률 워크스페이스입니다. 사건 중심 구조로 설계되어, 개별 사건에 관련된 문서·파일·대화·일정이 하나의 공간에서 연결됩니다.

## 시작하기

{% stepper %}
{% step %}
#### 회원가입 및 자격 인증

이메일 또는 소셜 계정으로 가입하고, 변호사 신분증 또는 직업 정보 서류를 제출합니다.
{% endstep %}

{% step %}
#### 심사 후 온보딩

변호사 신분 인증은 즉시 가능하며, 타 직업군은 영업일 기준 1–2일 소요됩니다. 승인 후 프로필 설정을 완료하면 워크스페이스에 진입합니다.
{% endstep %}

{% step %}
#### 사건 생성

새 사건을 생성하고 사건 정보를 등록합니다. AI가 이 정보를 기반으로 문서 작성과 검색을 지원합니다.
{% endstep %}

{% step %}
#### 법률 업무 혁신

채팅으로 AI와 대화하거나, 문서작성 탭에서 초안을 생성하고, 판례 검색으로 근거를 확보하세요.
{% endstep %}
{% endstepper %}
