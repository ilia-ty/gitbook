# 결재 관리

## 결재 상신

사건 관리에서 작성한 문서를 선택하고 결재 기안서를 작성한 뒤 결재 라인을 지정하여 상신합니다.

{% stepper %}
{% step %}
### 결재 상신

문서에서 결재 상신 버튼을 클릭하고 결재 라인과 기안 내용을 작성합니다.
{% endstep %}

{% step %}
### 결재자 검토

결재자는 문서를 확인하고 승인, 반려, 수정 요청을 처리합니다.
{% endstep %}

{% step %}
### 결과 통보

결재 완료 시 상신자에게 결과가 전달됩니다.
{% endstep %}
{% endstepper %}

## 나에게 온 결재

결재 요청이 들어온 문서 목록을 확인하고 승인, 반려, 수정 요청을 처리합니다.
